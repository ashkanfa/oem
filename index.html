Orthogonalizing EM

The orthogonalizing EM algorithm is an efficient method for the computation of penalized least squares problems. The algorithm works by

  1. augmenting the original design matrix such that the augmented matrix is orthogonal
  2. treating the augmented responses as missing
  3. use EM iterations with simple closed form expressions to solve original problem

The oem algorithm works for a variety of penalties including the lasso, elastic net, MCP, and SCAD.

Usage

n <- 5000
p <- 200
m <- 25
b <- matrix(c(runif(m, -0.5, 0.5), rep(0, p - m)))
x <- matrix(rnorm(n * p, sd = 3), n, p)
y <- drop(x %*% b) + rnorm(n)
fit <- oem(x, y, penalty = "lasso")
